# Layout grid

**UX-DS-LAYOUT-001 — Responsive progression.** Start one column; add columns only when content remains readable. Page-level horizontal scrolling is prohibited. Values for grid/gutter/width remain executable implementation detail.

**UX-DS-LAYOUT-002 — Reading layout.** Constrain public prose and preserve whitespace at larger widths rather than stretching long-form text. Reflow supports 200% zoom without task loss.

**UX-DS-LAYOUT-003 — Logical direction.** Use logical CSS properties/alignment. Mirror directional navigation, pagination, and breadcrumbs only. Do not mirror media, logos, code controls, or non-directional icons.

**UX-DS-LAYOUT-004 — Chrome and media.** Header, drawer, fixed action, and safe-area spacing preserve access to content. Repeated media has stable reserved area to avoid loading shift.

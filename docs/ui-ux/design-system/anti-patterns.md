# Anti-patterns

**UX-DS-ANTI-001 — Visual excess.** No gradients, glow, bokeh, glassmorphism, decorative blur, arbitrary elevation, oversized promotional type, or decorative choreography. Dark mode is not M1 scope.

**UX-DS-ANTI-002 — Structural excess.** No nested/card-stack pages, citation tiles, hidden essential mobile navigation, generic page renderer before repetition, or unsupported asset/content placeholder.

**UX-DS-ANTI-003 — Accessibility regression.** No removed focus indication, hover/gesture/colour-only meaning, placeholder-only form labels, inaccessible overlay, or fixed UI that obscures content.

**UX-DS-ANTI-004 — Locale and contract regression.** No physical directional styling where logical properties apply, Latin-only typography assumption, justified Persian prose, mirroring non-directional assets, slug/locale replacement, silent locale fallback, raw CMS HTML, or invented API/CMS facts.

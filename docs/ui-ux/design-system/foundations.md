# Foundations

**UX-DS-FOUND-001 — Direction.** Academic Editorial Utility is light-first, restrained, content-led, and professional. Canvas, typography, spacing, and hierarchy carry meaning before decoration. Dark mode is not approved for M1; see decision register.

**UX-DS-FOUND-002 — Surface restraint.** Prefer borders before elevation. Cards frame repeated records, dialogs, and tools; they do not frame every page section, prose paragraph, citation, or resume item.

**UX-DS-FOUND-003 — Semantic roles.** Canvas/surface, text, border, action/link, status, focus, spacing, type, radius, elevation, motion, and content-width roles express purpose, not component-specific styling. Functional colour is never sole state indicator.

**UX-DS-FOUND-004 — Icons.** Use one approved vector icon family with consistent hierarchy. Icon-only controls need accessible names. Primary/mobile navigation retains labels. Icon-family selection is UNRESOLVED; do not normalize an unapproved library.

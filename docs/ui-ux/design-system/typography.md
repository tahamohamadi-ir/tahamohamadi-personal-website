# Typography

**UX-DS-TYPE-001 — Locale hierarchy.** Runtime font stacks, scale, line height, and tracking are implementation-owned. English and Persian use an equivalent readable hierarchy; Persian does not inherit negative tracking or justified prose.

**UX-DS-TYPE-002 — Reading behaviour.** Long-form text uses constrained measure, wraps before truncation, and exposes complete text when truncation is unavoidable. Headings express document hierarchy, not promotional scale.

**UX-DS-TYPE-003 — Technical text.** Code, URLs, DOI values, email addresses, hashes, identifiers, and version strings remain LTR-isolated in Persian content. Citation punctuation and values remain copyable.

**UX-DS-TYPE-004 — Delivery gate.** Font hosting, licensing, subsetting, and production delivery are BLOCKED pending owner approval.

# Frontend

Initial Quasar 2 / Vue 3 frontend skeleton for TahaMohamadi.ir.

## Included

- Quasar with SSR configuration for public pages
- Vue Router routes for `/language`, `/fa`, `/en`, and `/admin`
- Pinia initialization
- `fa` and `en` message catalogs
- Public and admin layout placeholders
- RTL/LTR placeholders driven by public route metadata

## Commands

```powershell
npm install
npm run dev
npm run build
```

The development server URL is printed by Quasar after `npm run dev` starts.

## Current Boundaries

This skeleton has no backend integration, authentication, route guards, CMS content, or implemented public/admin pages. Test tooling is not initialized yet.

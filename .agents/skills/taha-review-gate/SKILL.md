# Taha Review Gate Skill

## Required Reading

- AGENTS.md
- .codex/code-review-checklist.md
- .codex/definition-of-done.md
- related .codex rules
- changed files

## Instructions

Review the change against:

1. Scope control
2. Architecture rules
3. Security rules
4. Database rules
5. i18n rules
6. SEO rules
7. Testing rules
8. Documentation updates
9. Overengineering risk
10. Token/context efficiency

## Output

- Pass/fail summary
- Critical issues
- Non-critical improvements
- Missing tests
- Missing docs
- Recommended commit message

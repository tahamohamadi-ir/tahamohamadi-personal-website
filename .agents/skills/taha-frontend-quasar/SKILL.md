## Instructions

1. Use Vue Single File Components.
2. Use Quasar components consistently.
3. Use Pinia only for shared state.
4. Public pages must support /fa RTL and /en LTR.
5. Public pages must be SEO-aware.
6. Add loading, empty, error, and success states.
7. Keep admin forms simple.
8. Do not hardcode CMS-managed content except temporary mock data.
9. Add or update tests where needed.
10. Keep changes focused.
